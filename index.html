Welcome to my attempt at a minimalist website for CS381's Assignment 2.

This site uses my personal CSS boilerplate (similar to a template) that I created last year called minimal.css in CS181.
It minimizes the work of creating a brand new stylesheet.
This way, you can focus more of your time on designing your webpage instead of looking up css selectors and attribute values.

Unfortunately, if you want to go back to a page, you would have to use the back button of your browser.
I don't quite know how to implement a JavaScript "back" button yet.
And that is exactly the kinds of skill I am looking forward to learning more about in CS381!
